# OpenSpec Directory

This directory contains structured specifications for the voding-hub project.

## Structure

```
openspec/
├── proposals/    # Feature proposals (proposal.md files)
├── specs/        # Approved specifications
└── logs/         # Execution logs and history
```

## Workflow

1. **Research** → Generate `proposals/proposal.md`
2. **Review** → Approve and move to `specs/`
3. **Implementation** → Execute based on specs
